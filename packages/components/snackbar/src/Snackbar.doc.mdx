# Snackbar

Display brief, temporary notifications. Meant to be noticed without disrupting a user's experience or requiring an action to be taken.

## Install

```sh
npm install @spark-ui/snackbar
```

## Import

```tsx
import { Snackbar } from "@spark-ui/snackbar"
```
