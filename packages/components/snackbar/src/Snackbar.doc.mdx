# Snackbar

Display brief, temporary notifications. Meant to be noticed without disrupting a user's experience or requiring an action to be taken.

## Install

```sh
npm install @spark-ui/snackbar
```

## Import

```tsx
import { Snackbar, addSnackbar } from '@spark-ui/snackbar'
```

## Props

<ArgTypes
  of={Snackbar}
  description="A container for queued snackbars. It should be placed at the root of the app."
  exclude={['position']}
  subcomponents={{
    'Snackbar.Item': {
      of: Snackbar.Item,
      description: "The component that display each snackbar's content.",
    },
  }}
/>
